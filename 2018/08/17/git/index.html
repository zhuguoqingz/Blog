git常用命令总结

Veröffentlicht am 2018-08-17 | Edited on 2018-09-25 | in 关于前端

查看信息

git branch
查看当前分支
git log
显示当前分支的版本历史查看时间
git status
显示有变更的文件
git log -5 - -pretty - -oneline
列出过去的5次提交
git branch -l
列出本地分支e
git branch -r
列出所有远程分支
git branch -a
列出全部分支(本地和远程分支)
git remote -v
显示所有远程仓库

分支

git branch name
新建一个本地分支但停留在当前分支
git checkout -b name
新建一个本地分支并切换到当前分支
git checkout -d name
删除当前本地分支
git pull [ remote ] [ branch ]
上传本地分支到远程仓库
git pull [ remote ] –force
强行推送当前分支到远程仓库，即使有冲突
git pull [ remote ] –all
推送所有的分支到远程仓库
